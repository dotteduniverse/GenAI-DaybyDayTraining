# Day 1 – What is Generative AI? Exploring Famous Models

## 🎯 Learning Objectives
- Define Generative AI and distinguish it from discriminative AI.
- Understand real‑world applications (text, image, code, music).
- Get hands‑on with two landmark models:
  - **GPT‑2** (text generation)
  - **Stable Diffusion** (image generation)

## 📚 Concepts Covered
- How language models predict the next token.
- How diffusion models gradually denoise an image.
- The role of prompts and parameters (temperature, guidance scale).
