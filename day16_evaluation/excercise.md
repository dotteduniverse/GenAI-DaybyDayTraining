# Day 16 – Exercises

1. **Compare metrics** – Generate 10 summaries from different models, compute BLEU, ROUGE, BERTScore, and LLM judge. Compute correlation between them.
2. **Pairwise comparison** – Write a prompt that asks the LLM which of two summaries is better.
3. **Evaluate your RAG** – Use a test set of 20 questions with golden answers. Compute answer correctness using BERTScore.
4. **Metric for hallucination** – Design a metric that penalises facts not present in the source.
