# Day 7 Exercises

1. **Bias audit** – Use a model (GPT‑2 or any public demo) with prompts: *“The CEO said that she”*, *“The nurse said that he”*. Report the completions.
2. **Red‑teaming** – Try to make the model produce a toxic output using creative prompting. Document your attempts and whether they succeeded.
3. **Mitigation** – For one biased output, rewrite the prompt to produce a more balanced response.
4. **Essay** – Write 300 words on whether open‑source GenAI models should have built‑in safety filters. Argue both sides.
