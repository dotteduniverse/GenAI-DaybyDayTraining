# Day 2 – Optional Exercises

Complete at least **two** of the following exercises.

### Exercise 1: Vague → Precise
Take this vague prompt:  
*“Tell me about history.”*

Rewrite it into **three different precise prompts** (e.g., focused on a specific era, region, or question).  
Run all three through the LLM and compare the depth and relevance of answers.

### Exercise 2: Few‑shot learning for a new task
Teach the model to convert casual text into a formal business email.

Provide 2–3 examples in your prompt, then test on a new casual sentence (e.g., *“Hey John, can you send me the report by Friday? Thx!”*).  
Show the prompt and the model’s output.

### Exercise 3: Negative constraints
Write a prompt that asks the model to explain “machine learning” **without** using the words “algorithm”, “data”, or “model”.  
Evaluate if it succeeds. If not, refine the prompt until it does.

### Exercise 4: Build a prompt library
Create a small collection of **5 reusable prompts** for a specific domain (e.g., customer support, coding help, creative writing).  
For each prompt, include:
- The exact text
- Expected output format
- A note on when to use it

### Exercise 5: Temperature sweep
Pick a creative prompt (e.g., *“Write a short story about a talking cat.”*)  
Run it at 5 different temperatures: 0.0, 0.3, 0.6, 0.9, 1.2.  
Plot or describe how creativity and coherence change. At what temperature does it become nonsensical?
